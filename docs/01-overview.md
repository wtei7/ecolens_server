# 1. 서비스 개요 (서버 파트)

## 서비스명

**EcoLens**

## 서비스 설명

EcoLens는 사용자가 플로깅을 시작하고 카메라를 켠 채 걸으면 GPS를 통해 이동 경로를 기록하고, AI가 카메라 화면 속 쓰레기를 자동으로 탐지하는 모바일 서비스다.

탐지된 쓰레기는 현재 위치와 함께 지도에 자동 등록된다. 사용자가 쓰레기를 주운 후에는 탐지 전후 이미지를 비교하여 정화 여부를 확인하고, 활동 종료 시 이동 거리와 정화 결과를 종합한 리포트를 제공한다.

## 핵심 사용자 경험 중 서버가 담당하는 부분

```text
플로깅 시작        → 세션 생성
GPS 이동 경로 기록 → 좌표 저장 (Batch API)
카메라 쓰레기 탐지 → AI 서버에서 탐지 결과 반환
쓰레기 위치 등록   → 쓰레기 기록 + 위치 저장
중복 탐지 방지     → 동일 쓰레기 판정 로직
쓰레기 수거        → Before·After 비교 검증
정화 여부 확인     → 상태 갱신 + 점수 지급
플로깅 종료        → 세션 종료 + 통계 집계
활동 리포트        → 리포트 데이터 생성
```

## MVP 목표 (서버 측)

백엔드는 다음 기능을 지원해야 한다.

1. 회원가입/로그인 및 JWT 발급
2. 플로깅 세션 생성·조회·일시정지·재개·종료
3. GPS 좌표 저장 (단건 + Batch)
4. AI 쓰레기 탐지 API (`/api/ai/detect`)
5. Before·After 정화 검증 API (`/api/ai/verify-cleaning`)
6. 쓰레기 기록 CRUD 및 중복 탐지 검사
7. 점수 계산
8. 환경지도 영역 기반 쓰레기 조회 + 클러스터링
9. 활동 리포트 사용자 통계

## MVP 제외 기능 (서버 파트)

- 친구 추가 / 실시간 경쟁
- 전국 랭킹
- 복잡한 업적 시스템
- 지자체 자동 신고
- AI 추천 플로깅 코스
- 불법 투기 예측
- 푸시 알림 (FCM 등)
- SNS 공유 이미지 생성

제외된 기능은 MVP 이후 확장한다.

## 탐지 대상 쓰레기 종류

- 캔 (`can`)
- 페트병 (`plastic_bottle`)
- 종이컵 (`paper_cup`)
- 비닐 (`vinyl`)
- 일반 쓰레기 (`general`)
- 유리병 (`glass_bottle`) — 점수 테이블에 명시됨
- 대형 쓰레기 (`bulky`) — 점수 테이블에 명시됨

## AI 모드

AI 모델이 준비되지 않은 초기 단계에서는 Mock AI API를 제공한다. 환경 변수로 모드를 전환한다.

```env
AI_MODE=mock   # mock | yolo
```

Mock 모드에서는 테스트용 쓰레기 탐지 결과를 반환하여 프론트엔드/백엔드 병렬 개발을 가능하게 한다.
